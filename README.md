# Cyber Security Lab Setup

## Objective

To build a safe and isolated cybersecurity lab environment for practicing penetration testing, networking, and Security Operations Center (SOC) concepts without exposing systems to the public internet.

---

# Lab Architecture

## Machines Used

### Kali Linux
- Used as the attacker machine
- Performed reconnaissance and security testing tasks

### Windows 10
- Used as the victim machine
- Simulated a client workstation environment

### Windows Server 2019 / 2022
- Used as a domain server and centralized log source
- Simulated enterprise network infrastructure

---

# Network Configuration

An Internal / Host-Only network was configured to create a secure and isolated lab environment.

## Purpose of Isolation
- Safe penetration testing practice
- Controlled network communication
- No direct internet exposure
- Realistic cybersecurity simulation
